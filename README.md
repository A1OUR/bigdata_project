Для установки и запуска необходимо:
1.	Установить и запустить Docker Desktop
2.	Клонировать репозиторий проекта
3.	Запустить в командной строке в корневой папке
```bash
docker compose up --build
```
чтобы собрать образ контейнера
4.	При последующем запуске достаточно запуска
```bash
docker compose up
```
